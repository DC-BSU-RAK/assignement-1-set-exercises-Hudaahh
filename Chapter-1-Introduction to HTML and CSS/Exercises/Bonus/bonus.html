HTML & CSS Quick-Reference Guide

A clean and professional overview of essential HTML tags and CSS properties

HTML Tags

<h1> to <h6>

This one is Used to define the HTML headings, from bigger (<h1>) to the smallest (<h6>).

<h1>Main Heading</h1>

<p>

This explain a paragraph of text

<p>This is a paragraph</p>

<a>

Here we Makes a hyperlink to another resource or webpage

<a href="https://example.com">Visit Example</a>

<img>

Embeds an image into the document.

<img src="image.jpg" alt="Description">

<ul>, <ol>, <li>

specify unordered and ordered lists with list items.


<ul>
    <li>Item One</li>
    <li>Item Two</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

<div>

A box for grouping elements and applying styles.

<div class="box">Content inside a box</div>

<span>

Inline box used to style part of the text.

<p>This is a <span class="highlight">highlighted</span> word.</p>

<table>, <tr>, <td>, <th>

this is to applied to display tabular data.


<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>24</td>
    </tr>
</table>

<form>, <input>, <button>

Used to collect user input and submit data.


<form action="/submit" method="post">
    <input type="text" name="name" placeholder="Your Name">
    <button type="submit">Submit</button>
</form>

CSS Properties

color

Sets the color of text.

color: #333;

background color

Here we apply the background color of an element

background color: #f9f9f9;

font-family

This used to Defines the font of the text.

font-family: 'Segoe UI', sans-serif;

padding

Adds space inside an element, around the content

padding: 16px;

margin

Adds some space outside an element, around the border

margin: 20px;

border

Insert a border around an element

border: 1px solid #ccc;

text-align

Here it used to operat the horizontal alignment of text

text align: center;

box-shadow

Here we include shadow effects to elements

box-shadow: 2px 2px 8px rgba(1,0,1,0.1);
